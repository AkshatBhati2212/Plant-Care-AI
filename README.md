
# PlantCare AI – Plant Disease Classification

PlantCare AI is a deep learning project that uses **transfer learning with MobileNetV2** to classify plant leaf diseases from images. The model is trained on the **New Plant Diseases Dataset** containing over **87,000 images across 38 classes**, enabling accurate identification of plant diseases affecting crops such as Apple, Corn, Grape, Potato, Tomato, and more.

## Project Objective

The goal of this project is to build an **efficient and accurate plant disease classification model** that helps farmers, agricultural experts, and gardeners detect plant diseases early and take preventive actions.

## Dataset

* **Dataset:** New Plant Diseases Dataset
* **Total Images:** 87,000+
* **Classes:** 38 plant disease categories
* **Plant Types:** Apple, Corn, Grape, Potato, Tomato, and others

## Model Used

* **MobileNetV2 (Transfer Learning)**
* Pre-trained convolutional neural network
* Faster training with improved accuracy
* Reduced computational cost

## Technologies Used

* Python

* MobileNetV2

## Project Workflow

1. Data Collection and Loading
2. Data Preprocessing and Image Resizing

4. Model Training and Validation
5. Model Evaluation
6. Disease Prediction from Plant Leaf Images

## Applications

* **Automated Agricultural Monitoring** – Real-time crop disease detection in farms.
